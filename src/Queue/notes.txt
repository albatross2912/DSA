Queue
-------
1. Queue is linear DS.
2. Queue Follows FIFO principle [first in first out ].
3. Operations
        |-Enqueue : insert an element into the queue
        |-Dequeue : Delete an element from the queue
4. All operations perform from Two end points.
    |-FRONT-> Delete an element from the FRONT end.
    |-REAR-> insert an element from the REAR end.
5. Conditions
    a. FRONT = REAR == -1 Queue is Empty
    b. FRONT == REAR, Single Element
    c. REAR == size-1 ,Queue is Full
